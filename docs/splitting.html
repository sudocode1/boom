Splitting Reference

Default
Function splitting: |
Parameter splitting: ?
+hello|n|rng?100

Conditionals (if etc)
Function splitting: /
Parameter splitting: @
if?hello?=?hi?+hello/n/+this is 17 + 49:/++17@+@49?otherwise if?hi?=?hi?+hi! this is the second statement./n/+here is a random number from 0 to 103:/rng@103

Loops
Function splitting: #
Parameter splitting: &
loop?+hi#n#rng:100

Var/Edit
Function splitting: >
Parameter splitting: $
var?hello?function?+hello>rng$100

While
Function splitting: %
Parameter splitting: :
while?hello?=?hi?+hello%rng:100
